# Release Alpha (MVP)

## Objectif
Version alpha fonctionnelle (MVP) : navigation + catalogue + fiche détail.

## Prérequis
- Java
- MySQL
- Maven

## Lancement
1. Démarrer MySQL
2. Configurer `application.properties`
3. Lancer :
    - via IntelliJ (Run)
    - ou `mvn spring-boot:run`

## Parcours MVP (à tester)
- Home : `/`
- Catalogue : `/shows`
- Fiche spectacle : `/shows/{id}`
- Connexion : `/login`

## Notes
Cette version alpha ne contient pas encore toutes les fonctionnalités finales (réservations, reviews, etc.).
